# 実装済みコマンド一覧

# /open

  • 新規作成
/open [ServerName] [Version] [GoogleDriveURL]

[ServerName] サーバーの名前

空白、全角文字、記号などは使わず英小文字、英大文字、英数字を使用してください

例: abc123


[Version] マインクラフトバージョン

対応バージョンに記載されているバージョンを指定してください

例: 1.17.1


[GoogleDriveURL] Google Driveの共有URL

作成した共有URLを指定してください


  • クイック新規作成
/open [ServerName] [Version] [flat | normal]

[ServerName] サーバーの名前

空白、全角文字、記号などは使わず英小文字、英大文字、英数字を使用してください

例: abc123


[Version] マインクラフトバージョン

対応バージョンに記載されているバージョンを指定してください

例: 1.17.1


[flat | normal] ワールドタイプ

flat : スーパーフラット
normal : 通常ワールド


  • セーブデータ読み込み
/open [ServerName]

[ServerName] サーバーの名前

すでに保存されているサーバーデータの名前を指定します

例: abc123


# /stop

  • すべてのサーバーを閉じる (強制)
/open all

  • 指定したポート番号にバインドされたサーバーを閉じる
/open [PortNumber]

[PortNumber] ポート番号

# /list

  • 起動中のサーバーを列挙する
/list u
  • セーブされたサーバーデータの名前を列挙する
/list s
  • 対応しているバージョンを列挙する
/list v

# めんどくさくなって実装してないコマンドたち

  • ヘルプ表示させたかった
/help c
