# サーバーを作成

# 0 ) 目次

1.Zipを作成する
2.GoogleDriveにアップロードする
3.コマンドを入力する

# 1 ) Zipを作成する

ObserverBotを用いてサーバーを作成するには専用アプリ (opens new window)を用いて特殊な形式のZipファイルを作成する必要があります。

これを確認してください。

# 2 ) GoogleDriveにアップロードする

Google Drive (opens new window)に1で作成したファイルをアップロードします。

1で作成したファイルをドラッグアンドドロップします

ファイルのアップロードが終了したら、一度F5キーを押しページをリロードしてください
リロードが終わるとアップロードしたファイルが表示されているのでそのボックスを右クリックしてください

右クリックして表示されたコンテックス内のリンクを取得をクリックします。

ポップアップ内の制限付きの項目をリンクを知っている人全員に必ず変更してください

変更が完了次第、リンクをコピーをクリックします

# 3 )　コマンドを入力する

新規作成コマンドを使用します

/open [サーバー名] [バージョン] [GoogleDriveURL]

コマンドをObserverBotのDMに送信するか、
ObserverBotが導入されているサーバーのテキストチャンネルどれかに送信してください。
