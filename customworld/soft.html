# McZipper4ObserverBot

McZipper4ObserverBot はObserverBotにてサーバーを作成するための
ワールドデータ プラグイン モッド server.properties をZipファイルにまとめるアプリです。

# 1 ) インストール

Githubのリリースページ (opens new window)にアクセスし

Assetsの一番上の項目のMcZipper4ObserverBot.zipをクリックしてください

ダウンロードしたZipファイルを解凍します

解凍してできたフォルダーの中にMcZiper4ObserverBot.exeを起動してください

WARNING

WindowsDefenderのセキュリティ警告(WindowsによってPCが保護されました)みたいなやつ出てきたら 詳細情報から実行すれば起動できます。
((((一応僕が作ったのでウイルス等は入ってないです。。。。))))

# 2 ) Zipを作成する

# 1 ) ワールドデータを指定する

配布ワールドデータフォルダーの一階層下にlevel.datファイルが存在する必要があります

イメージ図

  • (配布ワールドデータフォルダー)
    • level.dat
    • ...

WARNING

ワールドデータフォルダのフォルダ名に空白が入っていると読み込めない可能性があります。
配布ワールドを利用する場合、サーバー作成時に配布ワールド対応バージョンとサーバーバージョンを合わせてください。 正常に読み込めない可能性があります。

アプリのワールドデータタブの参照ボタンを押します

これは例としてworlddataフォルダーが配布ワールドデータフォルダーとします。
フォルダーを選択してフォルダーの選択ボタンを押します。

# 2 ) プラグイン、Modファイルの指定

上記と同じようにアプリの追加ボタンを押して追加してください
はい。

# 3 ) プロパティの指定

プロパティタブの下矢印を押すといろいろ出てきます。
説明も書いてあるので、はい。

# 4 ) Zipの作成

作成しましょう。
